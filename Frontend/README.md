# Product Inventory Manager 

## Objective CLO - 01
A full-stack web application built to perform basic CRUD operations using a REST API and a live cloud database. This is strictly an individual submission.

## Tech Stack
* **Frontend:** HTML, CSS, JavaScript (Fetch API) with custom dark UI theme.
* **Backend:** ASP.NET Core Web API (C#)
* **Database:** MongoDB Atlas (Cloud)

## Setup Instructions
1. Clone this repository to your local machine.
2. Navigate to the `InventoryAPI` folder in your terminal and run: `dotnet run`
3. Note the localhost HTTP port (e.g., http://localhost:5185).
4. Update the `apiUrl` variable in `Frontend/index.html` and `Frontend/form.html` to match the active port.
5. Open `index.html` in any web browser to view and interact with the application.
